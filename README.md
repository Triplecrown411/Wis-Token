# Wis-Token

Welcome to **Wis-Token**, a private project focused on token management 
and database development. This repository contains contracts, scripts, 
and other resources to help build and maintain the database.

---

## Repository Rules

1. **Respect the code** — Do not make unauthorized changes to critical 
files.  
2. **Branches** — Work on feature branches. Only merge into `main` after 
testing.  
3. **Pull requests** — All contributions must go through a pull request 
for review.  
4. **Commits** — Write meaningful commit messages. Sign commits when 
possible.  
5. **Security** — Do not expose private keys, passwords, or sensitive 
information.
